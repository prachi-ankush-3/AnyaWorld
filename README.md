# 🌸 About the Project

**AnyaWorld** is a beautifully designed storybook website dedicated to **Anya Forger** from **SPY × FAMILY**.

Instead of simply displaying text, the project presents Anya's journey in an immersive **digital book experience**, where every chapter is accompanied by stunning illustrations, smooth animations, elegant layouts, and a pleasant reading interface.

The goal is to make readers feel as though they are reading a real illustrated storybook.
